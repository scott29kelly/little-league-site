# Agent Instructions

## Project

This repository is a client-safe demo for a youth baseball league website. It should complement SportsConnect now and remain easy to adapt as PlayMetrics integration rolls out.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Vercel-compatible static deployment unless a backend is explicitly needed

## Rules

- Do not add pricing numbers, dollar amounts, package rates, or sponsor fees to public pages or documentation.
- Keep registration language clear: the website points families to the approved league platform rather than replacing it.
- Avoid official Little League, MLB, or team marks unless the client supplies rights-approved assets.
- Prefer readable, explicit code over clever abstractions.
- Use named exports for reusable components. Default exports are fine for pages and layouts.
- Do not introduce dependencies unless the existing stack cannot reasonably solve the problem.
- Preserve Windows/PowerShell compatibility in scripts and commands.
